## 📌 Proyecto
VidaPet es un sistema web de gestión veterinaria desarrollado con **Java Spring Boot** y **Thymeleaf**.

Permite gestionar:
- Mascotas 🐶🐱
- Propietarios 👤
- Veterinarios 👨‍⚕️
- Citas 📅
- Consultas 🩺
